# فن الـ Lazying (فن الكسل الاستراتيجي)

اصنع قيمة أكبر بعمل أقل. هذا المستودع يجمع أساليب وأدوات "الكسل الاستراتيجي":
- عوامل ذكاء اصطناعي وأتمتة لتقليل التكرار
- تعلّم لغات وأدوات تعلّم سلبي (بطاقات كلمات على E‑ink)
- أمثلة واقعية ومدونات فيديو (تطبيق 80/20 يومياً)

للتفاصيل الكاملة والقوائم، راجع النسخة الإنجليزية.
