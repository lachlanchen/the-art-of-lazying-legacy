# 懶人藝術（The Art of Lazying）

用更少的力氣創造更多價值。此處整理「策略性偷懶」的方法與工具：
- AI 代理與自動化流程（減少重複勞動）
- 語言學習與被動學習小工具（如 E‑ink 單詞卡）
- 真實案例與 Vlog（把 80/20 原則用在日常）

完整內容與專案列表請見英文版 README。
