# Lazying の作法（The Art of Lazying）

最小の労力で最大の価値を。ここでは「戦略的な怠け方」を実践するための方法と道具をまとめています：
- AI エージェントと自動化（単純反復を減らす）
- 言語学習と受動学習の小物（E‑ink 単語カードなど）
- 実例と Vlog（80/20 を日常に）

詳しいプロジェクト一覧は英語版 README をご覧ください。
