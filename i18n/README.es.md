# El arte de Lazying (The Art of Lazying)

Crear más con menos esfuerzo. Este repositorio reúne métodos y herramientas del “lazier estratégico”:
- Agentes IA y automatización (reducir la repetición)
- Aprendizaje de idiomas y herramientas de aprendizaje pasivo (tarjetas E‑ink)
- Casos reales y vlogs (aplicar el 80/20 a lo cotidiano)

Para la lista completa de proyectos, consulte el README en inglés.
