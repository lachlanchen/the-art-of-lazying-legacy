# 懒人艺术（The Art of Lazying）

用更少的力气创造更多价值。这里收集了“策略性懒惰”的方法、示例与工具：
- AI 代理与自动化工作流（减少机械重复）
- 语言学习与被动学习小道具（如 E‑ink 单词卡）
- 真实用例与 Vlog（把 80/20 原则用于日常）

完整内容与项目列表请见英文版 README。
